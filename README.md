# Speech Recognition using Python

## Description
This project demonstrates a basic speech recognition system that converts
spoken language from an audio file into written text using Python.

## Objective
The goal of this project is to transform an audio file containing speech
into text using a speech recognition module.

## Tools
- Python
- SpeechRecognition library
- Natural Language Processing (NLP)

## Academic Context
Developed as part of a Natural Language Processing (NLP) course.
